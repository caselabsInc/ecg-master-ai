# ECG-Master Mobile App Asset Pack

## Brand direction
Premium clinical AI identity: dark medical interface, electric cyan ECG trace, controlled cardiac red accent, structured diagnostic-flow feel.

## Positioning
Premium AI-assisted ECG interpretation and training app for clinicians, medical students, residents, and practitioners.

## Core files
- icon.png — 1024x1024 universal app icon
- ios-icon-1024.png — iOS App Store icon
- play-store-icon-512.png — Google Play icon
- adaptive-icon-foreground.png — Android adaptive icon foreground
- adaptive-icon-background.png — Android adaptive icon background
- splash-dark-1290x2796.png — dark splash screen
- splash-light-1290x2796.png — light splash screen
- wordmark-light-text.png — wordmark for dark backgrounds
- wordmark-dark-text.png — wordmark for light backgrounds
- play-store-feature-graphic-1024x500.png — Play Store feature graphic
- social-preview-1200x630.png — social/link preview
- onboarding-hero-1080x1920.png — onboarding/marketing hero
- logo-mark.svg — editable vector logo source
- brand-tokens.json — colors and brand metadata

## Suggested app theme tokens
- Background: #07111F
- Surface: #0E1A33
- Primary: #21D4FD
- Secondary: #2563EB
- Accent/Error/Cardiac: #FF3B5C
- Text: #F8FAFC
- Muted text: #94A3B8

## Usage note
Use the ECG trace as the primary identity element. Keep red as an accent, not the dominant color, to avoid an emergency/alarm-heavy look.
